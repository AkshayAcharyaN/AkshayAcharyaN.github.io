Featured Projects

Multi PDF RAG Application

Designed and implemented a Retrieval-Augmented Generation (RAG) approach to address inefficiencies in managing and retrieving information from extensive document repositories. This solution enables users to pose natural language questions and retrieve accurate, relevant content from documents quickly. Leveraging advanced NLP and LLM techniques, the application improves productivity by reducing the time spent on manual searches. Built using a robust technical stack that includes Python, LangChain, NLTK, Tesseract, FAISS for efficient indexing, and Streamlit for an intuitive user interface, the project significantly enhances workflow efficiency and information retrieval accuracy.

Technical Stack: Python, FAISS, LangChain, Google Generative AI Embeddings, LangChain Text Splitter

Railways Tender Price Prediction Using Machine Learning and Deep Learning

Developed a predictive solution to assist clients in securing contracts by offering competitive tender bids. Conducted comprehensive data analysis to identify factors influencing tender prices and implemented clustering techniques to uncover bidding patterns. Leveraged a Feed Forward Neural Network (FFNN) to accurately (87%) predict tender prices, enabling clients to bid competitively without risking overpricing or underpricing. This project improved decision-making, increased the likelihood of contract wins, and is published in IJISRT.

Technical Stack: Python, pandas, numpy, scikit-learn, Pipeline, RobustScaler, OneHotEncoder, ColumnTransformer, TensorFlow, Keras, FFNN

Deep Learning-Based Solar Panel Defect Detection

Developed a computer vision solution to improve the monitoring and maintenance of large photovoltaic (PV) systems. The project involved data collection and analysis to identify various defects occurring in solar plants. Leveraged drone-captured images to train a model capable of detecting defects with an accuracy of 89.8%, enabling faster and more accurate defect identification. This streamlined the maintenance process, reduced manual errors, and optimized resource utilization.

Technical Stack: Python, YOLO, PyTorch, TensorFlow, Roboflow, Streamlit

Dynamic Data Visualization and Analysis Chatbot for CXOs

Developed a CXO chatbot for a leading beverage company in Uttar Pradesh to address inefficiencies in retrieving insights from extensive CSV files. The chatbot integrates advanced technologies, including Gemini 2.0 Flash, GPT-4, DeepSeek R1, and CSV Agent, to provide rapid and accurate information retrieval. It enables CXOs to interact with data conversationally, generate insightful visualizations dynamically within the chatbot, and make informed decisions faster.

Technical Stack: Python, Google Generative AI (Gemini), PandasAI, Groq, LLaMA, FAISS, LangChain Prompt Engineering
